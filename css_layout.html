CSS layouts

CSS layouts er tekniken som ble oppfunnet for å kunne styre utsene og strukturere siden. Men det er egentlig bare navnet på forskjellige teknikker for sidestruktur. Inne disse så har du noen som f.eks Flexbox, Grid, Float og Frameworks.

Grid

Grid er en måte å sette opp siden sin i et rute nett og kan da sies å bli brukt i 2 dimensjoner samtidig, Loddrett og Vannrett. Eller som det heter for den "rows" og "collomns".

Flexbox

Flexbox er meget likt grid bare i den forstand at den kun kan brukes i en dimensjon. Den lager spaltene sine nedover eller bortover. Å bruke flexbox er en måte å sikkre seg at spaltene får like mye mellomrom mellom alle spaltene.
